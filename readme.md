## express-station
快速建站

## branch desc
+ base 底层分支，基于express，ejs框架系统
+ master 集合分支，所有类型分支的集合
+ station 后台界面分支，基本的后台界面，可以直接扩展开发
+ socket socket分支，使系统支持socket，已写入multiplexing例子

## dependencies
+ express
	+ 服务器框架
+ ejs
	+ 页面模板引擎
+ body-parser
	+ 网页参数解析器，req.query
+ express-params
	+ url匹配解析器，req.params.xxxx
